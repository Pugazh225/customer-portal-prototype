Dispensation Permit — Terms and Conditions

1. Application Terms
By submitting this dispensation application, you agree to abide by the terms set forth by the local authority for car park permit usage.

2. Payment
All fees are non-refundable once the permit has been issued. Cancellation before issuance may be eligible for a partial refund subject to council policy.

3. Duration & Bays
The permit is valid only for the specified bays and location. Any changes require a new application.

4. Non-Zonal Rules
This permit does not grant zonal access. The correspondence address provided is for communication purposes only.

5. Liability
The council is not liable for any issues arising from incorrect information provided in this application.

About This Application

This is a Dispensation (non-zonal / car-park) permit application. It requires a correspondence address instead of a zonal address.

Address Step

Select a saved address or enter one manually. All fields (Postcode, Property, Street, Town) are required for manual entry.

Bay Details

Select the street and add bay rows with type and number of spaces. "Reason for purchase" is mandatory. The number of bays is derived from the bay selection section.

Purchase Limits

Non-zonal permissions may have purchase limits. If the limit is exceeded, you will be blocked from completing the purchase.
